# 🐍 Python Essentials - Comandos Básicos, Intermediários e Avançados

Este repositório reúne dois notebooks desenvolvidos durante meus estudos da linguagem **Python**. O objetivo é servir como material de consulta e revisão, apresentando desde os conceitos fundamentais até recursos mais avançados da linguagem, acompanhados de exemplos práticos e comentados.

## 📚 Conteúdo

O repositório está organizado em dois notebooks:

### 📘 Notebook 1 — Fundamentos da Linguagem

Aborda os principais conceitos para quem está iniciando em Python.

**Tópicos:**

- Sintaxe básica
- Variáveis
- Tipos de dados
- Operadores
- Entrada e saída de dados
- Estruturas condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`for` e `while`)
- Funções
- Lambda Functions
- Strings
- Listas
- Tuplas
- Conjuntos (`set`)
- Dicionários
---

### 📗 Notebook 2 — Recursos Intermediários e Avançados

Apresenta funcionalidades utilizadas em aplicações mais robustas e no desenvolvimento profissional.

**Tópicos:**

- Parametros e Argumentos
- Escopos
- Return
- Empacotamento de argumentos (`args`)
- Manipulação de Dicionários
- List, Dictionary e Set Comprehension
- Iteradores
- Geradores (`yield`)
- Decoradores
- Programação Funcional
- Organização de Codigo
- Persistencia de dados
- Bibliotecas Padrão (`itertools`,`collections`,`os`,`pathlib`,`json`)

## 🚀 Como executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```

2. Entre na pasta do projeto:

```bash
cd nome-do-repositorio
```

3. Instale o Jupyter Notebook (caso ainda não possua):

```bash
pip install notebook
```

4. Execute:

```bash
jupyter notebook
```

ou utilize o **Visual Studio Code** com a extensão **Jupyter**.
